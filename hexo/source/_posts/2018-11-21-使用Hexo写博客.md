5. 使用hexo清除旧的静态页面
```
hexo clean
```

6. 使用hexo命令生成新的页面
```
hexo generate
```

7. 使用hexo命令本地测试
```
hexo server
```

8. 使用hexo命令部署
```
hexo deploy
```
